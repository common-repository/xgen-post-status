xGen Post Status helps the user to manage the status of the post by setting the status(active/inactive) permanently or for a specific period of time.

== Description ==

xGen Post Status provide options to the user whether to show/hide the post permanently or on temporary basis.

While developing the plugin, we were emphasizing on one thing that minimum settings should be required to make the plugin work. So after installation of the plugin, minimum configuration is required to make the plugin work.

If you have an idea which will make this plugin better then please do share it with us by emailing us on admin@xgensolutions.in.

We will implement your idea and release a new version for the same.

== Installation ==

1. Upload `xgen-post-status` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Then go to Post->Add New, below description you will see the xGen Post Status settings. 

== Frequently asked questions ==

**Q. What if I need to permanently set the status to inactive for a specific post?**
**Ans.** In this case, select the option Inactive from the drop-down box and keep "Start Date" and "End Date" field blank. Then click on publish to save the post which will also save the setting with it


**Q. What if I want to set the status of the post to inactive for a specific period of time?**
**Ans.**  Select the option Inactive from the drop-down box and select "Start Date" and "End Date" by selecting the date from datepicker. Then click on publish to save the post which will also save the setting with it
